# AI Chatbot

## Description
A simple AI Chatbot developed in Java. It takes user input and responds with predefined answers.

## Features
- Greets the user
- Responds to basic questions
- Uses Java Scanner for input
- Simple if-else logic

## Technologies Used
- Java

## How to Run
1. Open the project in VS Code or IntelliJ IDEA.
2. Compile Main.java.
3. Run the program.
4. Enter a message and get the chatbot response.
